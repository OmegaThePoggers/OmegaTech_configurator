## Data Flow

1. Application loads and fetches available parts from `components.json`.
2. User selects components via dropdowns; `script.js` updates total cost, performance score, FPS estimates, and 3D viewer.
3. User saves build; POST request is sent to `save_config.php`.
4. `save_config.php` appends the configuration to `configurations.json`.
