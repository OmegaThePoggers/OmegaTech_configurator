Migrate the content and structure of the legacy `index.html`, `about.html`, and `contact.html` pages into React server components within the new Next.js App Router structure, utilizing the established dark tech minimalist theme.

- src/app/page.tsx
- src/app/layout.tsx
- legacy/index.html
- legacy/about.html
- legacy/contact.html
